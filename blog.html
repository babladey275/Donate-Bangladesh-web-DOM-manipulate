Important FAQ

What is the Document Object Model (DOM)?

Document object model is a tree structure representing HTML documents, allowing to developer access, modify content, create/remove elements and handle events using JavaScript.

How do you select an element from the DOM?

Using the getElementById(), getElementsByClassName(), getElementByTagName(), querySelector() and querySelectorAll() methods.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a design pattern in the DOM used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a parent element. It reduces number of event listeners, dynamic handling and code simplicities.
